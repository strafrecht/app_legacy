Eine Fotokopie...

  • stellt immer eine Urkunde dar, weil sie im Rechtsverkehr als Ersatz für das Original praktisch anerkannt ist und ihr vergleichbares Vertrauen entgegengebracht wird.

Nach einer Mindermeinung stellt auch die Fotokopie heutzutage eine Urkunde dar, weil deren Verwendung im Rechts- und Geschäftsverkehr mittlerweile üblich ist (vgl. Freund JuS 1991, 723 ff).

  • stellt nur im Ausnahmefall eine Urkunde dar, wenn sie zum einen einer Originalurkunde so ähnlich ist, dass eine Verwechslung nicht ausgeschlossen werden kann, und der Täter zum anderen den Vorsatz hat, mit der Kopie den Anschein einer Originalurkunde (Urschrift) zu erwecken.

BGH und h.L. behandeln Fotokopien grundsätzlich wie Abschriften, also als Nicht-Urkunden. Denn der Aussteller eines Originals bekennt sich nicht zum Inhalt beliebiger Vervielfältigungen. Die schlichte Fotokopie weist lediglich auf das Vorhandensein eines Originals hin. Eine Ausnahme soll lediglich dann gelten, wenn die Kopie zum einen einer Originalurkunde so ähnlich ist, dass eine Verwechslung nicht ausgeschlossen werden kann, und der Täter zum anderen den Vorsatz hat, mit der Kopie den Anschein einer Originalurkunde (Urschrift) zu erwecken.

  • stellt nie eine Urkunde dar, weil es sich um eine bloße Reproduktion eines Originals handelt.
