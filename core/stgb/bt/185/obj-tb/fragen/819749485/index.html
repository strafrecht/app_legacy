Handelt es sich um eine Beleidigung, wenn der Täter die Äußerung in einer für den Adressaten fremden Sprache abgibt?

  • Der Täter hat die Äußerung ehrverletzend gemeint. Es spielt keine Rolle, ob der Adressat die Äußerung versteht.

Die sinnliche Wahrnehmung einer Äußerung allein genügt nicht.

  • Der Adressat versteht die Äußerung nicht, es liegt keine Beleidigung vor.

Der Adressat muss auch den Sinn der Äußerung erfassen, ansonsten kann keine Beleidigung vorliegen.

  • Es kommt allein auf die Tonlage an, in der die Äußerung getätigt wurde.
