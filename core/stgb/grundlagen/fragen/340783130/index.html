Der Erfolg der negativen Generalprävention kann ...

  • ... überhaupt nicht empirisch überprüft werden.
  • ... durch Befragungen untersucht werden.
  • ... anhand von Rückfallstatistiken überprüft werden.

Nach dem Strafzweck der negativen Generalprävention soll Strafe dazu dienen, andere von der Begehung von Straftaten abzuhalten. Ob ein solcher Erfolg gegeben ist, kann durch Befragungen untersucht werden.
